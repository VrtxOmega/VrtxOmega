## Doctrine

> *The pipeline does not determine what is true.*
> *It determines what survives disciplined attempts to falsify it.*

**Local-First** — Offline operation is the default wherever architecturally possible. Network access is scoped and opt-in.

**Deterministic Validation** — AI outputs pass a multi-gate cryptographic audit before acceptance. Unvalidated AI output is treated as an adversarial surface.

**Zero Telemetry** — No external analytics, no cloud callbacks, no third-party data capture.

**Operator Model** — The Cortex gate requires explicit operator approval before AI-generated actions execute.
